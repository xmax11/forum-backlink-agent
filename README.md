# Backlink Agent

A fully automated, multi‑forum backlink posting agent designed for real‑world SEO workflows.  
It handles crawling, anchor extraction, reply generation, account creation, captcha solving, and posting — all triggered from GitHub Actions or CLI.

---

## ✅ Features

- Multi‑forum support (XenForo, phpBB, Discourse, vBulletin, MyBB, SMF, IPS, Vanilla)
- Automated account creation using temp email + captcha solving
- Thread parsing and context‑aware reply generation
- Groq‑powered natural anchor insertion
- Duplicate‑post prevention
- Config‑driven architecture
- JSON‑based account and thread tracking
- Modular, production‑ready codebase
